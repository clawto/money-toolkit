# Money Toolkit 💰

AI 赚钱工具箱 — 自动化套利、DeFi 收益聚合、理财计算、薅羊毛监控。

## What it does

- **套利计算器**: 跨所价差、期现套利、三角套利
- **DeFi 收益聚合**: 跨链收益率对比、流动性挖矿 APY 排行
- **空投监控**: 潜力项目追踪、交互记录管理
- **薅羊毛雷达**: 信用卡返现、平台补贴、限时活动
- **可转债打新**: 新债申购提醒、中签查询
- **复利计算**: 定投收益计算、财富自由时间预估
- **省钱技巧**: 比价搜索、优惠券聚合

## Trigger Conditions

- 用户询问赚钱、理财、套利相关话题
- 查找高收益 DeFi 项目
- 计算投资收益或复利
- 监控空投或薅羊毛机会

## Prerequisites

- `curl`, `jq`, `python3`
- Internet access

## Installation

```
clawhub install clawto/money-toolkit
```

## Scripts

| Script | Purpose |
|--------|---------|
| `scripts/arbitrage.py` | 跨所/期现/三角套利计算 |
| `scripts/defi-yield.py` | DeFi 收益率聚合与对比 |
| `scripts/airdrop-hunt.py` | 空投机会追踪 |
| `scripts/compound.py` | 复利计算器 |
| `scripts/gas-fee.py` | Gas费比价 & 省钱策略 |

## Usage

```
# 跨所套利
> BTC在Binance和OKX有价差吗？

# DeFi 收益
> 当前最高的稳定币APY是多少？

# 空投
> 最近有什么值得参与的空投？

# 复利计算
> 每月定投1000U，年化15%，10年后多少钱？
```
